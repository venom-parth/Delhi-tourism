## 🌟 Overview

**Delhi.** is a premium, modern tourism application built to showcase the rich culture, deep heritage, and bustling urban landscape of Delhi, India. 

Designed with cutting-edge UI/UX principles, the application features an immersive **dark and light mode** experience, butter-smooth scroll animations powered by GSAP, and a custom-built interactive geographic map that responds to user telemetry.

## ✨ Features

- 🌓 **Dynamic Theme System:** A flawless Light / Dark mode toggle built with semantic CSS variables, ensuring high-contrast typography and adaptive glassmorphism over immersive background photography.
- 🗺️ **Interactive Geographic Map:** A mathematically accurate, scalable vector map of Delhi featuring precise bounds for districts, the Yamuna River, Metro lines, and heritage hotspots.
- 🎥 **Cinematic Animations:** Powered by **GSAP** (GreenSock), featuring a dynamic scroll progress bar, staggered typography reveals, and parallax scrolling backgrounds.
- 📱 **Fully Responsive:** Tailored layout for Mobile, Tablet, and Desktop resolutions using Tailwind CSS.
- 🎨 **Premium Aesthetic:** Immersive dark mode with neon saffron accents (`#FF6B00`), noise-texture overlays, and bespoke bento-grid layouts.
- 🏛️ **Rich Content:** Comprehensive guides detailing Monuments (Red Fort, Qutub Minar), Markets (Chandni Chowk), local Cuisine, and Climate info.

## 🚀 Quick Start

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) (v18+) and npm/yarn installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VenomDevX/Delhi-tourism.git
   cd Delhi-tourism
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. Open `http://localhost:3000` (or the port provided by Vite) in your browser.

## 🛠️ Technology Stack

- **Frontend Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4) with Custom Semantic CSS Custom Properties
- **Animations:** GSAP (GreenSock Animation Platform)
- **Icons:** Lucide React
